# Charts.css

Open source data visualization framework using pure CSS.

## Installation

Install with npm:

```shell
npm install charts.css
```

Install with yarn:

```shell
yarn add charts.css
```

## Getting started

Check the full documentation on [ChartsCSS.org](https://ChartsCSS.org/).
